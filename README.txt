Woocommerce product slider Addons for Visual Composer. To create amazing product carousel/slider this plugin will help you.

== Description ==

Addons for Visual Composer features professional looking, easy to use yet highly functional extensions that can be used in a WPBakery Visual Composer page builder. 
This is really a cool plugin if you are using woo-commerce and you can easly make your product slideshow using this extention you can get for free.

A few notes about the sections above:

*   "Visual Composer plugin" must be installed and activated to use this plugin. After you activate the required plugins, the elements should be available for use in Visual Composer.
*   "Woocommerce Plugin " Must Need install and activated to use this plugin.
*   "Tested up to" is the highest version that you've *successfully used to test the plugin*. Note that it might work on
higher versions... this is just the highest one you've verified.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wc_product_carousel.zip` to the `/wp-content/plugins/` directory (But must need to install woo-commerce and visual composer first.)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit any page with visual composer there you can see one category is added on visual composer.
